Notes

Shell Inheritance

All wrapping components inherit from lib/helpers/dds.component.ts; This base component includes default inputs such as:

  • elementId (optional) - If left unsupplied, a random ID will be used instead. This is applied to the base element of the DDS component and similar IDs are generated for all linked/child components.
  • init: "now" | "later" | "manual" = "now"; Instructs the shell component to initialize during AfterViewInit (now), or at some point via mutationObserver (later), or manually from the parent component (manual) by calling its method, initializeNow()

Initialization Timing

The component shell normally initializes the DDS component within the AfterViewInit life cycle stage. However if you add the init property to any of the components' instances, it will initialize AFTER that step, whenever the DDS component is found in the DOM.
